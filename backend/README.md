# AgilityFitTodayApp Backend
Backend Server built to try out building & Deploying REST APIs / GraphQL server

## Proposed Features

- Create Account
- Login
- Top Workouts

## Tutorials

### REST API example tutorial

[Tutorial](https://medium.com/the-dev-caf%C3%A9/build-and-deploy-a-scalable-graphql-api-d912df6b7c28)

Uses:
* NodeJS (runs server)
* Nodemon (for development to update quickly on file changes)
* MongoDB (database)
* ApolloServer (converts javascript string to graphQL schema)

### Apollo GraphQL Server Tutorial

[Tutorial](https://www.apollographql.com/docs/apollo-server/getting-started/)

Run `node apollo-graphql-tutorial/in
dex.js` to run local server.

Uses:
* ApolloServer
